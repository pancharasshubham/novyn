# NOVYN — Scope

## Goal

NOVYN helps me:

- search saved content
- revisit forgotten content
- understand recurring interests
- identify attention patterns
- understand what I repeatedly save

The first user is myself.

Success means:

> I repeatedly use NOVYN to search and revisit my saved content.

---

# V1 Features

## Import
- Instagram saved posts JSON

## Search
- search by keywords
- search creators
- search captions

## Timeline
- saved content over time

## Themes
- repeated topics
- common interests

## Insights
- interest evolution
- recurring behavior

## Revisit
- forgotten content
- old saves worth revisiting
