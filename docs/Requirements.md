# Functional Requirements

## Import

User uploads Instagram JSON.

System validates:
- file type
- structure
- supported format

---

## Search

User can search:
- keywords
- creator names
- hashtags

---

## Saved Items

Display:
- title
- creator
- date
- source
- tags

---

## Insights

Generate:
- top interests
- repeated themes
- timeline shifts

---

## Revisit

Show:
- old saves
- forgotten saves
- important saves

---

- fast

- responsive
